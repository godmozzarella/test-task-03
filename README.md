# Уютная кухня — лендинг

Vite + чистый HTML/CSS/JS.

## Запуск

```bash
npm install
npm run dev
```

## Структура

```
├── index.html       # вся разметка
└── src/
    ├── styles.css   # все стили
    ├── main.js      # точка входа: дёргает init-функции
    ├── slider.js
    ├── accordion.js
    └── booking.js
```
